# 📌 WindowTopmost — 窗口置顶工具

一个轻量级的 Windows 窗口置顶管理工具，让你把任意窗口钉在最前面，方便多窗口切换时快速参考内容。

## ✨ 功能

- **一键置顶** — 双击列表中的窗口即可切换置顶/取消置顶
- **实时窗口列表** — 自动枚举当前所有可见窗口，按置顶状态排序
- **关键词搜索** — 快速过滤定位目标窗口
- **自动刷新** — 默认每 3 秒自动刷新窗口列表，保持同步
- **一键取消全部** — 关闭程序时自动取消所有置顶，不留残留
- **Catppuccin 暗色主题** — 护眼深色 UI

## 🚀 使用

### 从源码运行

```bash
python window_topmost.py
```

无需安装任何第三方依赖，仅使用 Python 标准库 + ctypes（调用 Win32 API）。

> **要求**：Python 3.6+，Windows 系统

### 使用打包好的 EXE

直接双击 `WindowTopmost.exe` 即可运行，无需安装 Python。

## 🎮 操作说明

| 操作 | 说明 |
|------|------|
| **双击列表项** | 切换该窗口的置顶状态 |
| **📌 置顶/取消** | 对选中窗口切换置顶 |
| **🪟 显示该窗口** | 将选中窗口置前并激活 |
| **🔄 刷新** | 手动刷新窗口列表 |
| **🔓 全部取消置顶** | 一键取消所有已置顶的窗口 |
| **自动刷新** | 开启/关闭每 3 秒的自动列表刷新 |
